# FlyingBireGame
作为java面向对象编程课的项目作业进行尝试的一款小游戏

-----
##使用到的知识介绍  
-单例模式的时间得分  
-对象池模式的障碍物生成与回收  
-策略模式的道具效果（虽然道具效果大部分具体实现在其他类里而不在策略类里）  
-Java 2D API使用（很好用）  
-Java AWT 的双缓冲绘图  
-进程创立  
-按键监听  

##基于原游戏教程的改进之处  
-将障碍判定框的持续存在改为碰撞后消逝 并在重新取出对象池时归还判定框  
-使用纯粹的判断语句实现了暂停/主菜单/运行/结算/双击Esc退出的流畅运行  
-增加道具效果 及其外观显示  

##自我评价  
-除了道具的代码实现以外 大体符合SOLID原则  
-几乎无bug  
-AI不使用 导致游戏内容丰富性和效率不是很高  
-但都是自己写的 体会到java的美丽之处
